# 簡易アプリ仕様設計シート — bamiri — SHARE

| 項目            | 内容                           |
| --------------- | ------------------------------ |
| 製品名          | bamiri — SHARE                 |
| バージョン      | v2.1                           |
| 種別            | Web アプリ（SaaS）             |
| URL（本番想定） | Vercel デプロイ先              |
| 対象ユーザー    | 振付師・ダンスインストラクター |

## 目的

ダンスのフォーメーション（ステージ配置）を、カウント単位で編集し BPM で再生確認できる。共有リンクでメンバーに閲覧させる。

## 主要機能

| #   | 機能                 | 概要                                           |
| --- | -------------------- | ---------------------------------------------- |
| 1   | フォーメーション編集 | メンバーをステージ上に D&D 配置（最大 500 人） |
| 2   | タイムライン         | セクション・カウント・半カウント（&）          |
| 3   | BPM 再生             | テンポ同期で配置遷移を確認                     |
| 4   | ステージ描画         | 矢印・×・ペン（カウント単位）                  |
| 5   | プロジェクト管理     | 複数曲・フォルダー・ブックマーク               |
| 6   | 認証・同期           | Supabase Auth、クラウド同期                    |
| 7   | 共有                 | 閲覧専用 URL 発行                              |
| 8   | ASK AI               | Gemini による操作ヘルプ                        |
| 9   | 課金                 | Stripe Pro（月 500 円、無制限プロジェクト）    |

## 画面一覧

| 画面       | パス         | 説明                     |
| ---------- | ------------ | ------------------------ |
| エディター | `/`          | メイン編集画面           |
| ログイン   | `/login`     | 認証                     |
| マイページ | `/mypage`    | プロフィール・プラン     |
| 共有閲覧   | `/?shareId=` | 閲覧専用（ログイン不要） |

## 外部連携

| サービス             | 用途                 |
| -------------------- | -------------------- |
| Supabase             | 認証・DB・Storage    |
| Stripe               | サブスクリプション   |
| Google Gemini        | ASK AI               |
| Spotify / YouTube 等 | 音源リンクメタデータ |

## 制約・前提

- ブラウザ: モダン Chromium / Safari（**モバイル・PC 両対応**、`src/app/responsive.css`）
- リアルタイム共同編集: 非対応
- オフライン: localStorage のみ（未ログイン時）

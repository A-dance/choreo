# bamiri — SHARE 操作マニュアル

この文書はアプリの機能と操作方法の公式マニュアルです。ASK AI はこの内容を根拠に回答します。

---

## 1. アプリ概要

bamiri — SHARE は、ダンスのフォーメーション（ステージ上のメンバー配置）を BPM に合わせて編集・再生できる Web アプリです。

| 用語 | 意味 |
|------|------|
| メンバー（ドット） | ステージ上の色付きの丸 |
| カウント | タイムライン上の拍（1, 2, 3…） |
| 半カウント（&） | 拍と拍の間の半拍タイミング（移動だけ半拍） |
| セクション | イントロ・Aメロ・サビなどの曲の区切り |
| ばみり | ステージ上の方眼グリッド（0 がセンター） |

**アクセス**
- 通常は **ログイン** が必要です（`/login`）。
- **共有リンク**（`?share=` 付き URL）から開いた場合のみ、ログインなしで閲覧できます。

編集内容は **自動保存** されます。ログイン中はクラウドにも同期されます。

## 2. 画面構成

| 場所 | 内容 |
|------|------|
| 左上 | プロジェクトメニュー（≡）→ サイドバー |
| 上部ヘッダー | 曲名、BPM・ばみり・ドット、人数、Play / Copy / Paste / Undo / ASK AI / Share |
| 中央 | ステージ（BACK / AUDIENCE 表示あり） |
| 下部 | タイムライン（セクションタブとカウント） |
| サイドバー | プロジェクト一覧、**音源**・**参考動画**、マイページ（アバタークリック） |

プロジェクトが 0 件のときは、画面中央に **プロジェクトがありません** と **+ 新規プロジェクト** が表示されます。

## 3. ログイン・アカウント

### 3.1 ログイン・新規登録
- `/login` で **ログイン** / **新規登録** を切り替え
- メールアドレスとパスワード、または **Googleで続ける**
- 新規登録時は **表示名** と **パスワード（確認）** が必要（8 文字以上・大文字小文字を含む）
- 登録後、確認メールが送られる場合があります

### 3.2 パスワード再設定
1. ログイン画面の **忘れた方はこちら** をクリック
2. メールアドレスを入力して再設定メールを送信
3. メールのリンクから `/auth/reset-password` で新しいパスワードを設定

### 3.3 ログアウト・アカウント削除
- **マイページ**（サイドバー上部のアバター、または `/mypage`）から **ログアウト**
- **アカウントを削除** … 確認ダイアログのあと、クラウド上のデータも削除されます

### 3.4 マイページ
- **← エディターに戻る** で編集画面へ
- **表示名**・**メール**（読み取り専用）・**言語**（English / 日本語）を変更
- **アイコン画像** … 画像を選ぶ / 画像を削除
- **プラン** … 無料（プロジェクト 1 件）または Pro。無料時はアップグレード案内あり
- プロジェクト一覧（読み取り専用、**使用中** バッジ表示）

## 4. プロジェクト

### 4.1 サイドバーの開き方
- 左上 **≡** をクリック。背景クリックまたは **閉じる**（×）で閉じます。

### 4.2 切り替え・新規作成
1. サイドバーでプロジェクト名をクリックして切り替え
2. **+ 新規プロジェクト** でダイアログを開く
3. 入力項目: **曲名**、**BPM**、**人数**、**カウント数（各セクション）** → **作成**
4. 新規プロジェクトは初期セクション（イントロ・Aメロ・サビ・アウトロ等）付きで作成されます

### 4.3 並べ替え・削除
- プロジェクトが 2 件以上のとき、一覧を **ドラッグ** して並べ替え
- 各行右端の **×** で削除（確認ダイアログあり）
- 最後の 1 件を削除すると空の状態になり、中央に新規作成プロンプトが出ます
- 各行に **最終保存** の日時が表示されます

### 4.4 プラン制限
- **無料**: プロジェクト 1 件まで
- **Pro**（月額 500 円）: プロジェクト無制限・クラウド同期
- 2 件目を作ろうとすると **Proにアップグレード** ダイアログが表示されます（Stripe 決済は準備中）

---

## 5. ヘッダーツール（曲名の右側）

### 5.1 曲名
- ヘッダーの入力欄で編集（自動保存）

### 5.2 BPM
- **BPM** 欄（40〜240）。Enter または欄外クリックで確定

### 5.3 ばみり（方眼）
- **横** … 左右マス数（1〜20）、**縦** … 奥行き（1〜20）。0 がセンター
- 有効な数値を入力するとリアルタイムで反映されます

### 5.4 ドットサイズ
- **ドット** 欄でメンバー丸の表示サイズ（14〜64 px）
1. **ドット** 欄をクリック
2. 数値を入力
3. **Enter** または欄外クリックで確定

### 5.5 人数
- **人数** ボタン（数字が表示）→ **メンバーパネル**（§6）

### 5.6 ヘッダー操作ボタン（英語表記）
- **Play** … 再生 / 一時停止
- **Copy** / **Paste** / **Undo** … 配置のコピー・ペースト・元に戻す
- **ASK AI** … ヘルプチャット（? アイコン付き）
- **Share** … 共有ダイアログ

---

## 6. ステージ操作

### 6.1 配置
- メンバー（丸）を **ドラッグ** して移動（方眼にスナップ）
- 短いクリックで選択、約 8px 以上動かすとドラッグ扱い

### 6.2 選択・解除
- メンバーをクリックで選択
- 空白クリックまたは **Escape** で選択解除

### 6.3 現在カウントだけ非表示
- メンバー選択中に **Delete** / **Backspace**
- メンバーパネルの **非表示** ボタン

### 6.4 カウント移動
- タイムラインの数字をクリック
- セクションタブをクリック → そのセクション先頭のカウントへ
- **← →** または **[ ]**

### 6.5 ステージのリサイズ
- ステージにマウスを乗せると端にハンドル表示
- 右・下・右下角をドラッグして表示サイズを調整

### 6.6 再生中の挙動
- メンバーを触ると再生が止まります
- 非表示メンバーも再生中はアニメーションします（見た目は透明）

### 6.7 ステージラベル
- **BACK**（奥）、**AUDIENCE**（手前）、**STAGE LEFT** / **STAGE RIGHT**

---

## 7. メンバーパネル

**開き方**: ヘッダー **人数** ボタン

- タイトル: **👥 メンバー編集**
- 上部に **編集中の位置**（セクション名・カウント）を表示
- **×**、背景クリック、**Escape** で閉じる

### 7.1 人数変更
- **人数** 欄（1〜500）→ Enter または欄外クリック

### 7.2 名前変更
- 各カードの名前欄を編集 → Enter または欄外クリック

### 7.3 表示 / 非表示（現在カウントのみ）
- **非表示** … このカウントだけ隠す
- **表示** … 非表示にしたメンバーを再表示

### 7.4 リストから削除
- **削除** … **削除済み** 一覧へ移動
- 削除済みメンバーは **表示** ボタン（ツールチップ: リストに戻す）で復元

---

## 8. タイムライン・セクション・カウント

### 8.1 セクションタブ
- 画面下部のタブがセクション
- **クリック** … 選択し、そのセクション先頭カウントへ移動
- **ドラッグ** … 並べ替え
- **ダブルクリック** … 名前編集（× でセクション削除。2 件以上のときのみ）
- セクション削除時は確認ダイアログが出ます

### 8.2 セクションの追加
1. セクションタブ列の右端 **+ Add section** をクリック
2. 新セクションが末尾に追加（初期名は「セクション5」など）
3. ダブルクリックで名前を変更

### 8.3 カウント
- **クリック** … そのカウントへ移動
- **ダブルクリック** … × 表示 → クリックで削除（配置データあり時は確認）
- もう一度ダブルクリックで × を閉じる。**Escape** でも × を閉じられます
- 配置があるカウントには印（ドット）が付きます
- カウント行は横スクロール可能（空白をドラッグでもスクロール）
- 左端に **S1**, **S2** … とセクション番号ラベル

### 8.4 カウントの追加
- 選択中セクションのカウント列右端 **+ Add count**
- 1 セクションあたり最大 **64** カウント（上限でボタン無効）

### 8.5 半カウント（&）
- カウント間の **+** をクリック → **&** を挿入
- 最後のカウントの後にも **+** があります
- 拍数は増えず、移動タイミングだけ半拍になります

---

## 9. 再生

- ヘッダー **Play** または **Space** で再生 / 一時停止
- BPM に合わせてカウントが進み、メンバーが各配置へ移動
- 再生中にカウントをクリック → その位置から再開
- **Escape** … メンバー選択中は選択解除、それ以外は再生停止

---

## 10. コピー・ペースト・Undo

| 操作 | 方法 |
|------|------|
| コピー | **Copy** または **⌘C** |
| ペースト | **Paste** または **⌘V**（コピー後のみ有効） |
| 元に戻す | **Undo** または **⌘Z** |

対象は **現在のカウント** の全メンバー配置です。

---

## 11. キーボードショートカット

| キー | 動作 |
|------|------|
| Space | 再生 / 一時停止 |
| ← → または [ ] | 前 / 次のカウント |
| Delete / Backspace | メンバー選択時: 現在カウントから非表示 / 未選択時: カウント削除 |
| ⌘C / Ctrl+C | 配置コピー |
| ⌘V / Ctrl+V | 配置ペースト |
| ⌘Z / Ctrl+Z | Undo |
| Escape | 選択解除、× 閉じる、パネル閉じる、再生停止 |

※ 入力欄フォーカス中は一部無効です。

## 12. 共有（Share）

### 12.1 リンクの作成
1. ヘッダー **Share** をクリック
2. ダイアログが開くと自動でリンク作成（**リンク作成中…**）
3. URL が表示されたら **コピー**、または **LINE** / **X** / **メール** / **リンクコピー** / **その他**（端末の共有シート）で送付

### 12.2 共有される内容
- フォーメーション（配置・セクション・カウント）
- 音源・参考動画の **リンク**（メタデータ）
- アップロード済みの **ローカルファイル** はリンクに含まれない場合があります

### 12.3 プレビュー
- ダイアログ内 **閲覧モードでプレビュー** で、共有相手と同じ見え方を確認できます
- プレビュー中はヘッダーに **閲覧モードを終了** が表示されます

### 12.4 共有リンクを開いた人（閲覧専用）
- ログイン不要で閲覧のみ
- 編集・Share・メンバー変更などは **できません**
- **閲覧モードを終了** ボタンは **表示されません**（共有 URL からはエディタに戻れません）

---

## 13. メディア（サイドバー）

サイドバーから **音源** / **参考動画** を開きます（件数表示: なし / N曲 / N本）。

### 13.1 音源
- http(s) の **リンクを貼り付けて追加**（Smart link として登録）
- **おすすめ**: Spotify、Apple Music、YouTube Music、Linkfire（lnk.to 等）、TuneCore LinkShare、song.link など
- **ファイル共有便**（メガファイル便・ギガファイル便など）の URL も追加できますが、アプリ内で音楽を再生するのは主に Spotify / YouTube Music。共有便のリンクは **外部サイトを開く** 形になります
- 追加後、対応サービスはパネル内プレビュー、すべて外部リンクアイコンから開けます
- **削除**（ゴミ箱アイコン）で除去

### 13.2 参考動画
- **YouTube / Vimeo** の URL を **追加**（ファイル共有便の動画 URL は非対応）
- 一覧から動画を選択して再生
- **タイトル**・**メモ** を編集可能
- **削除** で除去

### 13.3 閲覧専用モード
- 閲覧・再生は可能、追加・削除は不可

---

## 14. ASK AI（ヘルプ）

- ヘッダーの **ASK AI** ボタン（? アイコン + ラベル）をクリック
- チャット形式で使い方を質問（**無料・Pro とも回数制限なし**）
- **よくある質問** チップ、**新しい会話** で履歴クリア
- **Enter** で送信、**Shift+Enter** で改行
- **Escape** でパネルを閉じる

---

## 15. 通知（トースト）

画面下部に短いメッセージが表示されることがあります。例:
- プロジェクト切り替え・作成・削除
- コピー / ペースト完了
- 元に戻しました
- 音源・動画リンク追加
- 共有リンク関連のエラー（見つかりません、サイズ超過など）

---

## 16. 閲覧専用モード（まとめ）

| 状況 | 編集 | 終了方法 |
|------|------|----------|
| 共有リンクから開いた | 不可 | 終了ボタンなし（閲覧のみ） |
| Share ダイアログのプレビュー | 不可 | **閲覧モードを終了** |

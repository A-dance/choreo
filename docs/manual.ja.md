# bamiri — SHARE 操作マニュアル

この文書はアプリの機能と操作方法の公式マニュアルです。ASK AI はこの内容を根拠に回答します。

---

## 1. アプリ概要

bamiri — SHARE は、ダンスのフォーメーション（ステージ上のメンバー配置）を BPM に合わせて編集・再生できる Web アプリです。

| 用語               | 意味                                       |
| ------------------ | ------------------------------------------ |
| メンバー（ドット） | ステージ上の色付きの丸                     |
| カウント           | タイムライン上の拍（1, 2, 3…）             |
| 半カウント（&）    | 拍と拍の間の半拍タイミング（移動だけ半拍） |
| セクション         | イントロ・Aメロ・サビなどの曲の区切り      |
| ばみり             | ステージ上の方眼グリッド（0 がセンター）   |

**アクセス**

- 通常は **ログイン** が必要です（`/login`）。
- **共有リンク**（`?share=` 付き URL）から開いた場合のみ、ログインなしで閲覧できます。

編集内容は **自動保存** されます。ログイン中はクラウドにも同期されます。

## 2. 画面構成

| 場所         | 内容                                                                                                                                                   |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 左上         | プロジェクトメニュー（≡）→ サイドバー                                                                                                                  |
| 上部ヘッダー | 曲名、BPM・ばみり・ドット、人数、Play / Copy / Paste / Undo / **Share** / **ASK AI**                                                                   |
| 中央         | ステージ（BACK / AUDIENCE 表示あり）。右上に **Tool** ボタン（描画ツール）                                                                             |
| 下部         | タイムライン（セクションタブとカウント）                                                                                                               |
| サイドバー   | 検索・新規プロジェクト・フォルダー作成、プロジェクト一覧（ブックマーク / フォルダー / その他）、**音源**・**参考動画**、マイページ（アバタークリック） |

プロジェクトが 0 件のときは、画面中央に **プロジェクトがありません** と **+ 新規プロジェクト** が表示されます。

## 3. ログイン・アカウント

### 3.1 ログイン・新規登録

- `/login` で **ログイン** / **新規登録** を切り替え
- メールアドレスとパスワード、または **Googleで続ける**
- 新規登録時は **表示名** と **パスワード（確認）** が必要（8 文字以上・大文字小文字を含む）
- 登録後、確認メールが送られる場合があります

### 3.2 パスワード再設定

1. ログイン画面の **忘れた方はこちら** をクリック
2. メールアドレスを入力して再設定メールを送信
3. メールのリンクから `/auth/reset-password` で新しいパスワードを設定

### 3.3 ログアウト・アカウント削除

- **マイページ**（サイドバー上部のアバター、または `/mypage`）から **ログアウト**
- **アカウントを削除** … 確認ダイアログのあと、クラウド上のデータも削除されます
- Pro 契約中にアカウント削除すると、サブスクリプションも解約されます（ダイアログに案内あり）
- Pro の解約のみ行う場合は **サブスクリプションを管理**（Stripe ポータル）を使います

### 3.4 マイページ

- **← 戻る** で編集画面へ
- **表示名**・**メール**（読み取り専用）・**言語**（English / 日本語）を変更
- **アイコン画像** … 画像を選ぶ / 画像を削除
- **プラン** … 無料（プロジェクト 1 件）または Pro（無制限）。無料時はアップグレード案内あり
- **サブスクリプションを管理** … Pro 利用中は Stripe の顧客ポータルへ（解約・支払い方法の変更）。アプリ内に解約ボタンはありません
- プロジェクト一覧（読み取り専用、**使用中** バッジ表示）

## 4. プロジェクト（サイドバー）

### 4.1 サイドバーの開き方

- 左上 **≡** をクリック。背景クリックまたは **閉じる**（×）で閉じます。

### 4.2 ツールバー（検索・新規・フォルダー）

サイドバー上部にまとまっています。

| 項目                   | 操作                                                                                             |
| ---------------------- | ------------------------------------------------------------------------------------------------ |
| **検索…**              | プロジェクト名・フォルダー名で絞り込み。フォルダー名に一致した場合は中のプロジェクトもすべて表示 |
| **+ 新規プロジェクト** | 新規作成ダイアログ（曲名・BPM・人数・カウント数）                                                |
| **フォルダーを作成**   | 新しいフォルダーを追加（初期名は「新しいフォルダー」）                                           |

検索中はプロジェクトの並べ替え（ドラッグ）は無効です。

### 4.3 一覧の構成

プロジェクト一覧は次のブロックに分かれます（区切り線付き）。

1. **ブックマーク** … ☆ 付きのプロジェクト、および ☆ 付きフォルダー（フォルダーごと中身を表示）
2. **フォルダー** … 各フォルダーとその中のプロジェクト
3. **その他** … フォルダーに入っていないプロジェクト

### 4.4 プロジェクトの切り替え・新規作成

1. プロジェクト行を **クリック** して切り替え（切り替え時のトースト通知はありません）
2. **+ 新規プロジェクト** → **曲名**・**BPM**・**人数**・**カウント数（各セクション）** → **作成**
3. 新規プロジェクトは初期セクション（イントロ・Aメロ・サビ・アウトロ等）付きで作成されます

### 4.5 プロジェクト名の変更

- **サイドバー** … プロジェクト名を **ダブルクリック** → 下線付き入力 → **Enter** または欄外で確定。**Escape** でキャンセル
- **ヘッダー** … 現在開いているプロジェクトの曲名欄を直接編集（自動保存）

### 4.6 並べ替え

- プロジェクトが 2 件以上で、検索欄が空のとき、行を **ドラッグ** して並べ替え

### 4.7 フォルダー

- **フォルダーを作成** で追加。フォルダー名を **クリック** してリネーム（Enter / 欄外で確定）
- **▸ / ▾** でフォルダーの開閉
- プロジェクト行を **ドラッグ** してフォルダー見出しにドロップ → そのフォルダーへ移動
- フォルダーから出す … **その他** セクション（見出しまたは空きエリア）へドラッグ
- フォルダー内のプロジェクトを **その他** の行にドロップしてもフォルダーから外れます
- フォルダー右端 **×** でフォルダー削除（中のプロジェクトは **その他** に残る。プロジェクト自体は削除されません）
- フォルダー見出しの **☆**（× の左）でブックマーク ON/OFF

### 4.8 ブックマーク

- プロジェクト行右端の **☆**（× の左）で ON/OFF
- ブックマークしたプロジェクト・フォルダーは **ブックマーク** セクションにも表示されます（通常の場所にも残ります）

### 4.9 削除

- プロジェクト行・フォルダー見出し右端の **×** → 確認ダイアログ → 削除
- 最後の 1 プロジェクトを削除すると空の状態になり、画面中央に新規作成プロンプトが出ます
- 各行に **最終保存** の日時が表示されます

### 4.10 プラン制限

- **無料**: プロジェクト 1 件まで
- **Pro**（月額 500 円）: プロジェクト無制限
- 2 件目を作ろうとすると **Proにアップグレード** ダイアログ → **Stripeで申し込む**
- ログイン中は **無料・Pro とも** クラウド同期されます（Pro 限定ではありません）

## 5. ヘッダーツール（曲名の右側）

### 5.1 曲名

- ヘッダーの入力欄で編集（自動保存）

### 5.2 BPM

- **BPM** 欄（40〜240）。Enter または欄外クリックで確定

### 5.3 ばみり（方眼）

- **横** … 左右マス数（1〜20）、**縦** … 奥行き（1〜20）。0 がセンター
- 有効な数値を入力するとリアルタイムで反映されます

### 5.4 ドットサイズ

- **ドット** 欄でメンバー丸の表示サイズ（14〜64 px）

1. **ドット** 欄をクリック
2. 数値を入力
3. **Enter** または欄外クリックで確定

### 5.5 人数

- **人数** ボタン（数字が表示）→ **メンバーパネル**（§6）

### 5.6 ヘッダー操作ボタン（英語表記）

- **Play** … 再生 / 一時停止
- **Copy** / **Paste** / **Undo** … 配置のコピー・ペースト・元に戻す
- **Share** … 共有ダイアログ（ASK AI の左）
- **ASK AI** … ヘルプチャット（? アイコン付き、枠線は人数ボタンと同じスタイル）

※ **閲覧専用モード**（共有リンク・プレビュー）では **Share** と **ASK AI** は使えません。

## 6. ステージ操作

### 6.1 配置

- メンバー（丸）を **ドラッグ** して移動（方眼にスナップ）
- 短いクリックで選択、約 8px 以上動かすとドラッグ扱い

### 6.2 選択・解除

- メンバーをクリックで選択
- 空白クリックまたは **Escape** で選択解除

### 6.3 現在カウントだけ非表示

- メンバー選択中に **Delete** / **Backspace**
- メンバーパネルの **非表示** ボタン

### 6.4 カウント移動

- タイムラインの数字をクリック
- セクションタブをクリック → そのセクション先頭のカウントへ
- **← →** または **[ ]**

### 6.5 ステージのリサイズ

- ステージにマウスを乗せると端にハンドル表示
- 右・下・右下角をドラッグして表示サイズを調整

### 6.6 再生中の挙動

- メンバーを触ると再生が止まります
- 非表示メンバーも再生中はアニメーションします（見た目は透明）

### 6.7 ステージラベル

- **BACK**（奥）、**AUDIENCE**（手前）、**STAGE LEFT** / **STAGE RIGHT**

### 6.8 ステージ描画ツール（矢印・×・ペン）

ステージ右上の **Tool** ボタン（ペンアイコン + 「Tool」ラベル）で、**現在のカウント** に矢印・×マーク・ペン線を描けます。描画はカウントごとに保存され、カウントを切り替えるとそのカウント用の描画が表示されます。

#### 開き方・閉じ方

1. ステージ右上 **Tool** をクリック → ステージ内右上にツールバーが開く
2. ツールバー右端 **×** またはもう一度 **Tool** をクリックで閉じる
3. 再生中は **Tool** ボタンが無効

#### ツール（アイコンのみ）

| アイコン | 操作                                                                 |
| -------- | -------------------------------------------------------------------- |
| 矢印     | ドラッグで始点→終点の矢印を配置                                      |
| ×        | クリックで × マークを配置                                            |
| ペン     | ドラッグで自由に線を描く（離すたびに新しい線。何本でも続けて描ける） |

同じツールをもう一度クリックすると、メンバー移動モードに戻ります。

#### 色

- ツールバー下部の **色スウォッチ** で描画色を選ぶ
- 新しく描く線・矢印・× は選択中の色になる

#### 選択・移動・サイズ変更

- ツールバーを開いた状態で、**ペン以外** のとき既存の描画をクリックで選択
- **中央をドラッグ** … 移動
- **矢印の端** / **× の先端付近** をドラッグ … サイズ変更
- ツールバー左の **ゴミ箱**（選択を消す）で削除
- **ペン** がオンのときは描画が優先され、他の描画の選択・削除はしにくい → 削除したいときはペンをオフにする

#### メンバー配置との関係

- ツールバーを閉じているとき、または描画ツールをオフにしているときは、いつもどおりメンバーをドラッグして移動できます
- 描画は **Undo** の対象外です（削除はゴミ箱ボタン）

#### 閲覧専用モード

- 共有リンク・プレビューでは **Tool** ボタンと描画ツールは **表示されません**

## 7. メンバーパネル

**開き方**: ヘッダー **人数** ボタン

- タイトル: **👥 メンバー編集**
- 上部に **編集中の位置**（セクション名・カウント）を表示
- **×**、背景クリック、**Escape** で閉じる

### 7.1 人数変更

- **人数** 欄（1〜500）→ Enter または欄外クリック

### 7.2 名前変更

- 各カードの名前欄を編集 → Enter または欄外クリック

### 7.3 表示 / 非表示（現在カウントのみ）

- **非表示** … このカウントだけ隠す
- **表示** … 非表示にしたメンバーを再表示

### 7.4 リストから削除

- **削除** … **削除済み** 一覧へ移動
- 削除済みメンバーは **表示** ボタン（ツールチップ: リストに戻す）で復元

---

## 8. タイムライン・セクション・カウント

### 8.1 セクションタブ

- 画面下部のタブがセクション
- **クリック** … 選択し、そのセクション先頭カウントへ移動
- **ドラッグ** … 並べ替え
- **ダブルクリック** … 名前編集（× でセクション削除。2 件以上のときのみ）
- セクション削除時は確認ダイアログが出ます

### 8.2 セクションの追加

1. セクションタブ列の右端 **+ Add section** をクリック
2. 新セクションが末尾に追加（初期名は「セクション5」など）
3. ダブルクリックで名前を変更

### 8.3 カウント

- **クリック** … そのカウントへ移動
- **ダブルクリック** … × 表示 → クリックで削除（配置データあり時は確認）
- もう一度ダブルクリックで × を閉じる。**Escape** でも × を閉じられます
- 配置があるカウントには印（ドット）が付きます
- カウント行は横スクロール可能（空白をドラッグでもスクロール）
- 左端に **S1**, **S2** … とセクション番号ラベル

### 8.4 カウントの追加

- 選択中セクションのカウント列右端 **+ Add count**
- 1 セクションあたり最大 **64** カウント（上限でボタン無効）

### 8.5 半カウント（&）

- カウント間の **+** をクリック → **&** を挿入
- 最後のカウントの後にも **+** があります
- 拍数は増えず、移動タイミングだけ半拍になります

---

## 9. 再生

- ヘッダー **Play** または **Space** で再生 / 一時停止
- BPM に合わせてカウントが進み、メンバーが各配置へ移動
- 再生中にカウントをクリック → その位置から再開
- **Escape** … メンバー選択中は選択解除、それ以外は再生停止

---

## 10. コピー・ペースト・Undo

| 操作     | 方法                                        |
| -------- | ------------------------------------------- |
| コピー   | **Copy** または **⌘C**                      |
| ペースト | **Paste** または **⌘V**（コピー後のみ有効） |
| 元に戻す | **Undo** または **⌘Z**                      |

対象は **現在のカウント** の全メンバー配置です。

## 11. キーボードショートカット

| キー               | 動作                                                            |
| ------------------ | --------------------------------------------------------------- |
| Space              | 再生 / 一時停止                                                 |
| ← → または [ ]     | 前 / 次のカウント                                               |
| Delete / Backspace | メンバー選択時: 現在カウントから非表示 / 未選択時: カウント削除 |
| ⌘C / Ctrl+C        | 配置コピー                                                      |
| ⌘V / Ctrl+V        | 配置ペースト                                                    |
| ⌘Z / Ctrl+Z        | Undo                                                            |
| Escape             | 選択解除、× 閉じる、パネル閉じる、再生停止                      |

※ 入力欄フォーカス中は一部無効です。

## 12. 共有（Share）

### 12.1 リンクの作成

1. ヘッダー **Share** をクリック
2. ダイアログが開くと自動でリンク作成（**リンク作成中…**）
3. プロジェクトが複数ある場合:
   - **フォルダー** … 「すべての曲」「未分類」、または各フォルダーで絞り込み
   - **共有する曲** … 絞り込み後の一覧から共有するプロジェクトを選択
4. URL が表示されたら **コピー**、または **LINE** / **X** / **メール** / **リンクコピー** で送付

※ フォルダー行から直接共有するボタンはありません。必ず **Share** ダイアログから選びます。

### 12.2 共有される内容

- フォーメーション（配置・セクション・カウント）
- **ステージ描画**（矢印・×・ペン線。カウントごとのデータ）
- 音源・参考動画の **リンク**（メタデータ）
- アップロード済みの **ローカルファイル** はリンクに含まれない場合があります

### 12.3 プレビュー

- ダイアログ内 **閲覧モードでプレビュー** で、共有相手と同じ見え方を確認できます（プレビューする曲は **共有する曲** の選択に連動）
- プレビュー中はヘッダーに **閲覧モードを終了** が表示されます
- プレビュー中は **ASK AI** は使えません

### 12.4 共有リンクを開いた人（閲覧専用）

- ログイン不要で閲覧のみ
- 編集・Share・メンバー変更などは **できません**
- **閲覧モードを終了** ボタンは **表示されません**（共有 URL からはエディタに戻れません）

---

## 13. メディア（サイドバー）

サイドバーから **音源** / **参考動画** を開きます（件数表示: なし / N曲 / N本）。

### 13.1 音源

- http(s) の **リンクを貼り付けて追加**（Smart link として登録）
- **おすすめ**: Spotify、Apple Music、YouTube Music、Linkfire（lnk.to 等）、TuneCore LinkShare、song.link など
- **ファイル共有便**（メガファイル便・ギガファイル便など）の URL も追加できますが、アプリ内で音楽を再生するのは主に Spotify / YouTube Music。共有便のリンクは **外部サイトを開く** 形になります
- 追加後、対応サービスはパネル内プレビュー、すべて外部リンクアイコンから開けます
- **削除**（ゴミ箱アイコン）で除去

### 13.2 参考動画

- **YouTube / Vimeo** の URL を **追加**（ファイル共有便の動画 URL は非対応）
- 一覧から動画を選択して再生
- **タイトル**・**メモ** を編集可能
- **削除** で除去

### 13.3 閲覧専用モード

- 閲覧・再生は可能、追加・削除は不可

---

## 14. ASK AI（ヘルプ）

- ヘッダーの **ASK AI** ボタン（? アイコン + ラベル）をクリック
- チャット形式で使い方を質問（**無料・Pro とも回数制限なし**）
- **よくある質問** チップ、**新しい会話** で履歴クリア
- **Enter** で送信、**Shift+Enter** で改行
- **Escape** でパネルを閉じる

**使えないとき**

- **閲覧専用モード**（共有リンクから開いたとき、Share のプレビュー中）では **ASK AI** は無効です

---

## 15. 通知（トースト）

画面下部に短いメッセージが表示されることがあります。例:

- プロジェクト作成・削除
- コピー / ペースト完了
- 元に戻しました
- 音源・動画リンク追加
- 共有リンク関連のエラー（見つかりません、サイズ超過など）

※ プロジェクト切り替え時のトーストは表示されません。

---

## 16. 閲覧専用モード（まとめ）

| 状況                         | 編集 | Share | ASK AI | 終了方法                   |
| ---------------------------- | ---- | ----- | ------ | -------------------------- |
| 共有リンクから開いた         | 不可 | 不可  | 不可   | 終了ボタンなし（閲覧のみ） |
| Share ダイアログのプレビュー | 不可 | 不可  | 不可   | **閲覧モードを終了**       |
